React + Vite -> Frontend Website 
A professional, visually distinctive website that tracks upcoming and trending video games, with a hard focus on titles that matter to Indian and broader Asian gamers — not a generic "top 10 games" listicle. Think: a cross between an esports intel dashboard and a game-journalism outlet. The audience is competitive/casual gamers in India and Asia who care about battle royale titles (BGMI, Free Fire), anime gacha RPGs (Genshin, Wuthering Waves), and the big global AAA releases that still dominate conversation here (GTA 6).
